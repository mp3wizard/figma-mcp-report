Figma Official MCP — 17 → 18 tools

เพิ่ม download_assets — ดาวน์โหลด asset (รูป/ไอคอน) จากไฟล์ Figma ออกมาเป็นไฟล์แบบ batch ปิดช่องว่างเดิมที่มีแต่ upload_assets ขาเข้าอย่างเดียว ตอนนี้ asset ไหลได้สองทางครบ — มีประโยชน์ตรงกับงาน design-to-code handoff ที่ต้องดึง icon set / ภาพประกอบเข้า repo

Figma Console MCP — v1.29.0 → v1.31.0 (tools คงที่ 108)

รอบนี้ไม่มี tool ใหม่ แต่แก้ pain point อันดับหนึ่งของ Desktop Bridge: อาการ "connection หลุดแล้วต้อง restart plugin / Claude / kill port เอง" — v1.31.0 เพิ่ม self-healing connection (กำจัด zombie process อัตโนมัติ SIGTERM → SIGKILL + กวาดทุก 5 นาที), plugin auto-reconnect (probe ทุก 12 วิ ต่อกลับเองเมื่อ server โผล่), ปุ่ม Pause / Resume / Reconnect ใน plugin UI ใหม่ และ badge นับจำนวน server ที่ต่ออยู่ — ต้อง re-import plugin manifest ใน Figma Desktop ถึงจะได้ UI ใหม่

Framelink · Claude Talk to Figma · Sunnyside — ไม่มีฟีเจอร์ใหม่

Framelink ยัง v0.12.0 / 2 tools (stars ขยับ ~14,800 → ~14,984) · ClaudeTalk ยัง v1.0.0 / 93 tools (stars 561 → 595, ไม่มี release ใหม่ตั้งแต่ เม.ย.) · Sunnyside ยัง 27 tools, ไม่มี commit ใหม่ (stars 37 → 35)
